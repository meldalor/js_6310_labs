# Photo Gallery Demo

Демонстрационное приложение для библиотеки компонентов Photo Gallery UI.

## Описание

Веб-приложение демонстрирует использование компонентов Gallery и PhotoCard из библиотеки @photo-gallery/ui-library. Приложение отображает галерею из 6 фотографий с адаптивной версткой.

## Возможности

- ✅ Адаптивная галерея изображений
- ✅ Карточки фотографий с метаданными (название, описание, автор, дата)
- ✅ Responsive дизайн для разных размеров экранов
- ✅ Плавные анимации при наведении

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

### Сборка для продакшена

```bash
npm run build
```

### Предварительный просмотр продакшн сборки

```bash
npm run preview
```

## Скриншоты

### Десктоп

Галерея на больших экранах отображается в 3-4 колонки для удобного просмотра.

### Планшет

На планшетах галерея адаптируется до 2 колонок.

### Мобильные устройства

На мобильных устройствах галерея отображается в одну колонку.

## Особенности реализации

### Адаптивность

Галерея автоматически подстраивается под размер экрана благодаря CSS Grid и медиа-запросам:

- Мобильные (< 640px): 1 колонка
- Планшеты (640px - 1024px): 2 колонки
- Десктопы (1024px - 1440px): 3 колонки
- Большие экраны (> 1440px): 4 колонки

### Анимации

Карточки имеют плавные эффекты при наведении:
- Подъем карточки (translateY)
- Усиление тени
- Smooth transitions

### Типизация

Все компоненты полностью типизированы с помощью TypeScript, что обеспечивает:
- Автодополнение в IDE
- Проверку типов на этапе компиляции
- Лучшую документацию кода
