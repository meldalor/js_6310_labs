# Разработать компоненты:
1. Поле для игры в змейку
2. Компонент для отображение "змеи"
## Функциональные требования
- Поле принимает в себя размер и список из компонентов "змей", у которых есть информация об их расположении в пространстве
- Змея "знает" свое расположение в пространстве и свою конфигурацию
## Нефункциональные требования
- Запрещено использование сторонних UI библиотек
