# Разработать компонент Таблица - Список образовательных курсов
## Функциональные требования
- Таблица показывает названия курсов, описание, текущией статус обучения, имеет возможность фильтрации по параметрам
- При клике отображает информацию о курсе
## Нефункциональные требования
- Запрещено использование сторонних UI библиотек
