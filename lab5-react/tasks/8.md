# Разработать компонент Popup menu
## Функциональные требования
- Отображается при нажатии на элемент в котором находится
- Каждый элемент меню может быть как ссылкой так и кнопкой
- Корректно отображается при движении элемента на странице в случае ресайза окна
## Нефункциональные требования
- Запрещено использование сторонних UI библиотек
