# Разработать компоненты: 
1. Таблица карточек слов
2. Карточка для изучения
## Функциональные требования
1. Таблица карточек слов
- Таблица слов с переводами, при нажатии на слово отображается более подробная информация
2. Карточка для изучения
- Показывает слово
- Показывает контекст использования слова
- При нажатии на кнопку отображает перевод слова и перевод контекста использования
## Нефункциональные требования
- Запрещено использование сторонних UI библиотек
