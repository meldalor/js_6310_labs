# Разработать компонент Навигационное меню
## Функциональные требования
- Навигационное меню имеет единый стиль, расположено сверху или сбоку (в зависимости от настройки)
- Каждый элемент меню может быть ссылкой или кнопкой (в зависимости от настройки)
- Корректно работает при ресайзе страницы
## Нефункциональные требования
- Запрещено использование сторонних UI библиотек
