# Разработать компонент Универсальная таблица
## Функциональные требования
- Таблица отображает список колонок, список строк в зависимости от передаваемых параметров
- Имеет возможность пагинации
- Имеет возможность фильтрации
## Нефункциональные требования
- Запрещено использование сторонних UI библиотек
